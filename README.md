Baishi-mobile-new 
==========
**Baishi-mobile-new ** experiencing the best of new features about this new and definitive release: Angular 7 + Ionic 4 final. 

* Angular 7.2.2
* ionic 4.12.0
* Ionic Native 4
* Ionic 4 Components/API Samples
* 6 Page Samples (8 with modals)
* 2 Page modals
* 2 Components
---

Requirements
------------

* Node 10.13.5+
* Npm 6+
* Ionic CLI 4.12.0

Installing
------------

```
$ npm install
$ ionic serve
```
